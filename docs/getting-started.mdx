# Getting Started

## What is Slow React?

A collection of React performance problems for you to diagnose and fix using React Developer Tools.

## Prerequisites

- Basic knowledge of React
- React Developer Tools installed in your browser
- Node.js installed

## How it works

Each exercise contains a component with a real performance problem. Your job is to:

1. Open the app and interact with it
2. Open React DevTools → Profiler tab
3. Record and identify the problem
4. Fix it using the hints provided

## Tools you'll need

- **React DevTools** — to profile components and find re-renders
- **Chrome DevTools** — Performance tab for deeper analysis
- **Your editor** — to apply the fix

## Exercises

Start with the first exercise and work your way through. Each one builds on concepts from the previous.
