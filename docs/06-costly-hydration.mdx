## What is happening

This component imports `HeavySection` directly, which means it gets bundled and
loaded together with everything else on the initial page load. Open the Network
tab in Chrome DevTools and reload the page — everything loads in a single
chunk, even sections the user may never scroll to.

## Hints

- Direct imports are always included in the main bundle
- `lazy()` tells React to load a component only when it is needed
- `<Suspense>` is required to wrap any lazy component — it shows a fallback while loading
- Open the Network tab after applying the fix and look for a separate chunk
